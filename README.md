# 3-DoF
Software for 3 Degree of Freedom


To run create a virtual environment

``` python -m venv myenv ```

Install requirements.txt

``` pip install -r requirements.txt ```
 
 run main2.py

 use build.spec file to create exe

 ``` pyinstaller build.spec```

 For Installer run installer.iss by pressing ctrl + F9

 open the .iss file using Inno setup

 modify accordingly to include the correct paths
